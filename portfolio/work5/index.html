Vintage UI / Video Project: EqualTo 30

April’s fools project - Porting modern software (EqualTo) to Windows 3.11 and presenting it as a tech infomercial.

For EqualTo’s April 1st post I thought of creating a little alternate history by bringing our software back to 1992. The main sources of humor was based on:

  • How certain services would have made no sense in 1992 technology (think of the ‘Instagram in the 80s’ kind of videos)
  • The charm of outdated tech and tech jargon (Floppy diskettes)
  • Infomercial cringe (Call 1-800-INCENTIVIZE)

The app was designed in Visual Basic 3.0, in native Windows 3.11 / IBM ThinkPad 380XD. I’ve sketched up a few app designs, and I’ve eventually learned that the key to invoke a vintage feel is to exaggerate the style of the time period. Most Windows software from 1992 looked much more modern than we think of it. The intentionally clumsy use of big buttons, random unexplained color choices and system fonts, together with the possible lowest resolution helped to get the point across that this is vintage software.

The final script was written after having the software ready, and fun fact: about 2/3rd of the content was discarded before production. I’ve stitched my colleague’s voice clips together to a continuous voiceover and recorded the video to match it. For this, the software was running in a DosBox Win3.11 install, having the CPU turned down as low as possible in order to enhance those retro screen refresh artifacts.

In post production I wanted to add a CRT effect which was easily achieved by a few filters:

  • Color correction (shifted towards blue/magenta)
  • Some vertical banding to simulate a CRT grid (most similar to Trinitrons’ phosphor bars )
  • Horizontal blur and horizontal picture shift. Both of these should be very subtle, best if barely noticeable. They simulate scanlines of CRT screens.

Overall I’m happy with the outcome and the feedback was positive, however I didn’t have time to correct mistakes regarding the time period, eg. that some of those companies haven’t existed in 1992.
