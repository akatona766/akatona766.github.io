Vintage UI / Video Project: EqualTo 30

April’s fools project - Porting modern software (EqualTo) to Windows 3.1 and presenting it as a tech infomercial.

The concept

For EqualTo’s April 1st post I thought of creating a video from a parallel universe where our software has already existed back to 1992. The main sources of humor were:

  • How our tech would have made little sense as ‘cloud’ wasn’t really a thing yet
  • Outdated tech buzzwords like ’the information age’
  • Infomercial cringe (Call 1-800-INCENTIVIZE)

Software development

The app was a real working dummy software running on Windows 3.1. I’ve created multiple multiple designs actually, the one I’ve settled on uses the most exaggerated vintage design elements which would have been outdated even in 1992, for example using the System font, double borders and unexplained color choices. Manwhile the layout still reflected our modern software’s layout.

Script and voiceover

The final script was written after having the software ready and settling on a vague storyline. About 2/3rd of the original content was discarded before production, only keeping the best parts.

My colleague recorded voice clips that I’ve stitched together into a continuous voiceover and recorded the video to match it. For this, the software was running in a DosBox Win3.11 install, having screen resolution and CPU speed set as low as possible in order to create retro screen refresh artifacts.

Post production

In post production I’ve added a CRT effects using:

  • Color correction
  • Vertical banding to simulate a CRT grid (most similar to Trinitrons’ phosphor bars )
  • Horizontal blur and horizontal picture shift. Both of these should be very subtle. They simulate scanlines of CRT screens.

Conclusion

Overall I’m happy with the outcome and the feedback was positive.

However I didn’t have time to correct mistakes regarding the time period, eg. that some of those companies haven’t existed in 1992. Next time I’d pay more attenation to historical accuracy.
